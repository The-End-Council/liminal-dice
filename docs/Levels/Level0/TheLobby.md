# LEVEL0：The Lobby

## レベル情報

| 項目 | 内容 |
|---|---|
| 名前 | `LEVEL0：The Lobby` |
| 生存難易度 | `Class1` |
| レベルの概要 | 黄色い壁紙、湿ったカーペット、蛍光灯が続く迷路型のLevel |
| レベルの異常 | 視界外で構造が変化し、暗闇区画や赤い部屋が現れる |
| レベルの構造 | 屋内。水エリアは無。部屋、廊下、階段、穴、柱部屋、アーチ部屋で構成される |
| ゲーム化案 | 12面ダイス。通常エリア、Blackout Zone、Red Roomsの階層内移動を入れ、移動先で専用ダイスを振る |
| 出現エンティティ | Howler |
| 入口 | ゲーム開始時に到達する初期Level |
| 出口 | 穴に降りる、または点滅する壁を抜けてLevel 1へ移動する |
| 取得オブジェクト | アーモンドウォーター、乾パン |

### Blackout Zoneダイスイベント

Blackout Zoneは暗闇で方向を失う危険区画。
脱出か悪化を6面ダイスで判定する。

| 出目 | 行動 | 結果 | 状態 |
|---:|---|---|---|
| 1 | 粗い壁を手探りで進む | 精神力-5 | 未実装 |
| 2 | 足元の液体を踏み抜く | スタミナ-10 | 未実装 |
| 3 | 蛍光灯の音を追う | 通常エリアへ戻る | 未実装 |
| 4 | 微かな光へ走る | スタミナ-5。通常エリアへ戻る | 未実装 |
| 5 | 暗闇の奥の声を聞く | 精神力-10。もう一度Blackout Zoneダイスを振る | 未実装 |
| 6 | 壁の切れ目を見つける | アーチ部屋へ階層内移動する | 未実装 |

### Red Roomsダイスイベント

Red Roomsは閉じた赤い区画。
長居すると精神力を大きく削る危険エリアとして扱う。

| 出目 | 行動 | 結果 | 状態 |
|---:|---|---|---|
| 1 | 赤い壁紙を見続ける | 精神力-15 | 未実装 |
| 2 | 粘つくカーペットに足を取られる | スタミナ-15 | 未実装 |
| 3 | 入口へ引き返す | 通常エリアへ戻る | 未実装 |
| 4 | 剥がれた壁紙の裏を見る | 精神力-10。Liquid Painを発見する可能性がある | 未実装 |
| 5 | 閉じた廊下を一周する | Red Roomsに留まり、もう一度Red Roomsダイスを振る | 未実装 |
| 6 | 薄い黄色の光を見つける | Blackout Zoneへ階層内移動する | 未実装 |

## レベルの異常

| 異常 | 内容 | ゲーム化案 |
|---|---|---|
| Peripheral Shift | 視界外の部屋や通路が変化する | ダイス結果や移動先が変わるイベントにする |
| Blackout Zone | 照明が消えた暗闇区画 | 階層内移動先として扱い、脱出まで6面ダイスを振る |
| Red Rooms | 赤い色に包まれた危険区画 | 階層内移動先として扱い、精神力低下と閉じ込めを判定する |
| 湿ったカーペット | 正体不明の液体を含む床 | スタミナ消費やアイテム発見に使う |
| Isolation Effect | 複数人で入っても合流しづらい | 個別行動の不安演出に使う |
| 異常音 | 蛍光灯、声、壁の奥の音が聞こえる | 精神力低下やエンティティ遭遇の前兆にする |
