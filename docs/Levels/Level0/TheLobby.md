# LEVEL0：The Lobby

## レベル情報

| 項目 | 内容 |
|---|---|
| 名前 | `LEVEL0：The Lobby` |
| 生存難易度 | `Class1` |
| レベルの概要 | 黄色い壁紙、湿ったカーペット、蛍光灯が続く迷路型のLevel |
| レベルの異常 | 視界外で構造が変化し、暗闇区画や赤い部屋が現れる |
| レベルの構造 | 屋内。水エリアは無。部屋、廊下、階段、穴、柱部屋、アーチ部屋で構成される |
| ゲーム化案 | 12面ダイス。探索、アイテム取得、エンティティ遭遇、Level移動、精神力低下イベントを中心にする |
| 出現エンティティ | Howler |
| 入口 | ゲーム開始時に到達する初期Level |
| 出口 | 穴に降りる、または点滅する壁を抜けてLevel 1へ移動する |
| 取得オブジェクト | アーモンドウォーター、乾パン |

## レベルの異常

| 異常 | 内容 | ゲーム化案 |
|---|---|---|
| Peripheral Shift | 視界外の部屋や通路が変化する | ダイス結果や移動先が変わるイベントにする |
| Blackout Zone | 照明が消えた暗闇区画 | 懐中電灯がない場合は精神力を減少させる |
| Red Rooms | 赤い色に包まれた危険区画 | 高リスクの精神力低下イベントにする |
| 湿ったカーペット | 正体不明の液体を含む床 | スタミナ消費やアイテム発見に使う |
| Isolation Effect | 複数人で入っても合流しづらい | 個別行動の不安演出に使う |
| 異常音 | 蛍光灯、声、壁の奥の音が聞こえる | 精神力低下やエンティティ遭遇の前兆にする |
