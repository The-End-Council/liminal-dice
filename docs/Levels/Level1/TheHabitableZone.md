# LEVEL1：Habitable Zone

## レベル情報

| 項目 | 内容 |
|---|---|
| 名前 | `LEVEL1：Habitable Zone` |
| 生存難易度 | `Class1` |

| 取得オブジェクト | アーモンドウォーター、乾パン（実装済み）。改訂案では包帯・電池・防具素材・防具本体を追加する |

## レベルの異常

| 異常 | 内容 | ゲーム化案 |
|---|---|---|
| Flickering | 照明が断続的に消える現象 | エンティティ遭遇率上昇イベントに使う |
| Cratesの出現・消失 | 視線外で物資箱が現れたり消えたりする | アイテム獲得イベントの核にする |
| 距離感の錯乱 | ホール区画で距離感が狂いやすい | 探索失敗・再配置演出に使う |
| 低い霧 | 視認性を下げる環境要素 | 視界不良による遭遇前兆演出に使う |

## ゲームバランス方針

| 項目 | 方針 |
|---|---|
| アイテム供給 | Gild Sectorで供給を増やすが、消耗イベントで純増を抑える |
| 戦闘頻度 | Ouroboros / Garden Sectorで遭遇率を上げ、通常区画の事故率を下げる |
| 回復量 | Level 1では単発大回復を減らし、小中回復を増やす |
| 防具 | 事故死を減らすが、軽減上限と耐久で無敵化を防ぐ |
