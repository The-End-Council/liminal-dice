# Achievements：実績情報

## 実装済み実績

| ID | 名前 | 解除条件 |
|---|---|---|
| `firstExplore` | 初探索 | 初めて探索を行う |
| `firstEntity` | 初遭遇 | 初めてエンティティに遭遇する |
| `survivor` | 生存者 | Level 0をクリアする |
| `descend` | 降下者 | Level 1に到達する |
| `hunter` | エンティティハンター | エンティティを倒す |

## 保存方式

| 項目 | 内容 |
|---|---|
| 保存先 | `localStorage` |
| 保存キー | `backrooms-trophies` |
| 表示場所 | トロフィー・バッジ画面 |
