# GameSystem：ゲームシステム

## 概要

| 項目 | 内容 |
|---|---|
| ジャンル | ローグライク脱出ゲーム |
| テーマ | Backrooms階層探索 |
| プレイヤー人数 | 1人から4人 |
| 行動方式 | ダイスによるランダムイベント |
| 実行方法 | `Backrooms.html` をブラウザで開く |
| セーブ方式 | `localStorage` |

## 基本ステータス

| ステータス | 初期値 | 最大値 | 内容 |
|---|---:|---:|---|
| 体力 | 100 | 100 | 0になると行動不能 |
| スタミナ | 100 | 100 | 0になると行動不能 |
| 精神力 | 100 | 100 | 時間経過で減少する場合がある |
| 攻撃力 | 1 | なし | 武器や職業で増加する |

## ターンシステム

| 項目 | 内容 |
|---|---|
| 個別行動 | 各プレイヤーが個別にダイスを振る |
| 団体行動 | 全員で共通のダイスを振る |
| 個別行動コスト | スタミナ10消費 |
| 団体行動コスト | スタミナ5消費 |
| スタミナ0 | 体力が残っていても行動不能 |
| スタミナ不足確認 | 1人でもスタミナ0がいる場合、回復確認画面を表示 |
| 全員行動不能 | ゲームオーバー |

## ダイスシステム

| Level | ダイス | 内容 |
|---|---:|---|
| Level 0 | 14面 | 探索、アイテム取得、エンティティ遭遇、Level移動、階層内移動、ステータス変動 |
| Level 1 | 10面 | 探索、アイテム取得、エンティティ遭遇、Level移動 |

## 時間システム

| 項目 | 内容 |
|---|---|
| 開始時刻 | 06:00 |
| 経過時間 | 1ターンごとに6時間 |
| 24時間経過 | 次の06:00到達時に、行動不能でないプレイヤーの精神力が10減少 |
| 携帯バッテリー | 24時間経過ごとに1減少 |
| 時計表示 | 画面上に常時表示 |
| 時刻表示条件 | 屋内は時計系アイテム所持時のみ表示、未所持は`??:??` |
| 屋外表示条件 | 屋外Levelは時計系アイテム未所持でも時刻表示 |
| 時間帯表示 | 06:00は朝、12:00は昼、18:00は夜、24:00は深夜、未判明時は`?` |

## 戦闘システム

| 項目 | 内容 |
|---|---|
| 発生条件 | Levelイベントでエンティティに遭遇する |
| 攻撃 | プレイヤーの攻撃力と装備武器でダメージを与える |
| 逃走 | エンティティごとの逃走ダイスで判定する |
| 逃走コスト | 成功、失敗ともにスタミナ10消費 |
| 回避 | エンティティごとの攻撃ダイスで判定する |
| Howler攻撃ダイス | 6面: 4面が攻撃、2面が回避（面は毎回ランダム） |
| Howler逃走ダイス | 6面: 4面が逃走、2面が逃走失敗（面は毎回ランダム） |
| Duller攻撃ダイス | 6面: 4面が攻撃、2面が回避（面は毎回ランダム） |
| Duller逃走ダイス | 6面: 4面が逃走、2面が逃走失敗（面は毎回ランダム） |
| 全員体力0 | ゲームオーバー |

## 装備システム

| 項目 | 内容 |
|---|---|
| 武器装備 | バッグ内の武器をプレイヤーに装備できる |
| 装備中 | バッグから武器が外れる |
| 装備解除 | 武器がバッグへ戻る |
| ナイフ | 耐久を消費する |
| ハンドガン | 弾丸を消費する |

## バッグシステム

| 項目 | 内容 |
|---|---|
| 最大枠 | 12 |
| 同一アイテム | 数量としてまとめる |
| アイテム使用 | 回復系アイテムを対象プレイヤーに使用できる |
| 救護員補正 | パーティーに救護員がいる場合、HP回復とスタミナ回復の効果が10%上昇 |
| 満杯時取得 | バッグ満杯時は、捨てて取得するか確認する |
| アイテム破棄 | 個数を指定して捨てられる |

## セーブ・ロード

| 項目 | 内容 |
|---|---|
| セーブ数 | 3スロット |
| 保存先 | `localStorage` |
| 保存内容 | プレイヤー、バッグ、現在Level、時間、ターン、実績など |
| ロード | 現在の進行を置き換える |
| 削除 | スロット単位で削除できる |

## 実装済みLevel

| Level | 名前 | 説明 | 屋外 | 水辺 |
|---|---|---|---|---|
| Level 0 | The Lobby | 無限に広がる黄色い壁紙と湿ったカーペットの部屋。蛍光灯が点滅している。 | 無 | 無 |
| Level 1 | The Habitable Zone | コンクリートの廊下。かつて何かが存在したような痕跡が残っている。 | 無 | 無 |

## 実装済み演出

| 種類 | 内容 |
|---|---|
| BGM | ホーム、Level 0、Level 1 |
| SE | クリック、ダイス、アイテム、遭遇、戦闘、逃走、Level移動、実績、ゲームオーバー |
| 音量設定 | ホーム画面の「07 設定」でBGMとSEを別スライダー、別ミュートで調整できる |
| ゲームオーバー | 全画面オーバーレイと演出 |
